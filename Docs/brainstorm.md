# Pomodoro timer project
## Idea
I want to build a Pomodoro timer. The project is supposed to be released in steps (versions) to add features incrementally rather than developing everything at once. 

## Version ideas
1. I am going to start with MVP (must have): 
    - Countdown display
    - Actions: start, pause, reset
    - Work/Break mode
2. Version 2 (probable): 
    - Custom durations
    - Timer notification
3. Version 3 (very hypothetical at this point): 
    - Specific tasks log (eg. Create a design for Invoice page, Read that article, Organize workspace, etc)
    - Potentially connection to a Notion database to log history of tasks 
    - End of working session to celebrate a win
    - Extra polish 

## Thinking about MVP
A pomodoro timer, at its simplest, needs to:

- Display the time remaining in a countdown (minutes and seconds)
- Let the user control the timer: start, pause, and reset
- Track the current mode: are we in a work session or a break?
- Switch modes when a session ends: work → break, break → work
